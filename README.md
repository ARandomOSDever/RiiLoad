# RiiLoad
This is a Rust recreation of the [Wiiload](https://github.com/devkitPro/wiiload) tool, which is used to remotely load executables on Wii and Wii U consoles over the network.

## Non-features
 - USBGecko support: I don't have a normal Wii (only a Wii U) so I can't test it and I also don't think it's used that often. Support will be added if people ask for it
